virtual_composable Module¶

class roco.derived.composables.virtual_composable.VirtualComposable¶

Bases: roco.api.composable.Composable

Virtual Composable is an interface for objects which produce nontangible outputs.

Any class meant to produce nontangible output usable for the final robot designs should derive from this class. In any derived class, all member functions may be redefined.
